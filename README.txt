Breeze
Created by Christopher W. Johnson
Copyright (c) 2012 Silent Orb


Breeze is an HTML5 animation framework.  It includes the Breeze engine and Breeze Animator, an app for creating animations.

The Animator is designed to run inside the browser, allowing for cross-platform support and full WYSIWYG.

The Engine source is written in Javascript, while the Animator source was originally written in Javascript but is being converted to Coffeescript.
